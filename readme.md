# shift_cipher tool

A tool try to recover shifted cipher generated by some codes like Caesarâ€™s cipher, ROT13, shift-by-N, etc.  

Usage:
```rust
cargo run -p shift_cipher -- -h
```

Decrypt:
```rust
cargo run -p shift_cipher -- -d -c "OVDTHUFWVZZPISLRLFZHYLAOLYL"
```

Encrypt:
```rust
cargo run -p shift_cipher -- -e -p "howmanypossiblekeysarethere" -k 7 -o
```
